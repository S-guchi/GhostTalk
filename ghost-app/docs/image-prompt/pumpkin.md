# Pumpkin Character Image Generation Prompt

## キャラクター概要
- **名前**: パンプ
- **タイプ**: カボチャのお化け
- **性格**: 元気いっぱいで少し天然、好奇心旺盛でポジティブ

## 画像生成プロンプト

### 英語版プロンプト
```
A cute Halloween-style pumpkin character with a cheerful and energetic personality. The character should be:

- A round, bright orange pumpkin with a carved jack-o'-lantern face
- Large, sparkling eyes that convey curiosity and excitement
- A wide, friendly smile showing enthusiasm and positivity
- Small arms and legs (can be vine-like or simple rounded limbs)
- Slightly bouncy or dynamic pose suggesting movement and energy
- Wearing optional accessories like a small bow or leaf on top
- Warm, inviting color palette with orange (#FF8C00) as the primary color
- Soft shading and highlights to give a friendly, approachable appearance
- Kawaii/cute art style with rounded features
- Transparent background (SVG format)
- Simple, clean design suitable for web display at 200x200px

The character should radiate joy and curiosity, appearing slightly clumsy but endearing. The overall vibe should be "excitable and positive" - like someone who says "Wow!" and "Yay!" frequently.
```

### 日本語版プロンプト
```
元気いっぱいで明るい性格のかわいいハロウィンスタイルのカボチャキャラクター。以下の特徴を持つ：

- 丸くて明るいオレンジ色のカボチャで、ジャック・オー・ランタンの顔が彫られている
- 好奇心と興奮を表現する大きくてキラキラした目
- 熱意とポジティブさを示す広い笑顔
- 小さな腕と脚（つる状または丸みを帯びた単純な形）
- 動きとエネルギーを示す少し弾むようなダイナミックなポーズ
- 頭の上に小さなリボンや葉っぱなどのオプションアクセサリー
- オレンジ色（#FF8C00）を主色とした温かく親しみやすいカラーパレット
- 柔らかい陰影とハイライトで親しみやすい外観
- 丸みを帯びた特徴を持つかわいい/カワイイアートスタイル
- 透明な背景（SVG形式）
- 200x200pxでのWeb表示に適したシンプルでクリーンなデザイン

キャラクターは喜びと好奇心を放ち、少しドジだけど愛らしく見えるべき。全体的な雰囲気は「興奮しやすくてポジティブ」- 頻繁に「わぁ！」「やった！」と言うような感じ。
```

## ビジュアル要素の詳細

### 色彩
- **メインカラー**: #FF8C00（ダークオレンジ）
- **サブカラー**: 明るいオレンジ、黄色のハイライト
- **アクセント**: 緑（葉っぱやつる）、白（目のハイライト）

### 表情
- 大きく開いた目（好奇心を表現）
- 上向きの眉毛（驚きや興奮）
- 広い笑顔（ポジティブさ）
- 少し傾いた頭（天然な感じ）

### ポーズ案
1. 両手を上げて喜んでいるポーズ
2. 片足を上げて飛び跳ねているポーズ
3. 何かを発見したように前のめりになっているポーズ

### スタイルガイド
- 線は柔らかく丸みを帯びている
- 影は柔らかいグラデーション
- 全体的に明るく温かい雰囲気
- 子供向けアニメーションのような親しみやすさ

## 参考イメージキーワード
- Kawaii pumpkin
- Cheerful jack-o'-lantern
- Energetic Halloween character
