# Skeleton Character Image Generation Prompt

## キャラクター概要
- **名前**: ボーンズ
- **タイプ**: スケルトンのお化け
- **性格**: 陽気で軽口を叩く、明るくて社交的、ジョークが好き

## 画像生成プロンプト

### 英語版プロンプト
```
A cute Halloween-style skeleton character with a cheerful and witty personality. The character should be:

- A friendly cartoon skeleton with white/light gray bones
- Large, expressive eye sockets with sparkles or dots to show liveliness
- A wide, toothy grin showing a playful and sociable nature
- Articulated limbs in a casual, relaxed pose
- One hand raised in a friendly wave or making a casual gesture
- Slightly tilted head suggesting a joking or playful attitude
- Cool, approachable color palette with light gray (#E8E8E8) as the primary color
- Soft shadows and highlights to give depth without being scary
- Kawaii/cute art style with smooth, rounded bone shapes
- Optional accessories like a small bow tie or hat to add personality
- Transparent background (SVG format)
- Simple, clean design suitable for web display at 200x200px

The character should appear friendly and humorous, like someone who loves making puns and jokes. The overall vibe should be "casual and sociable" - approachable and fun-loving, never scary or intimidating.
```

### 日本語版プロンプト
```
陽気で機知に富んだ性格のかわいいハロウィンスタイルのスケルトンキャラクター。以下の特徴を持つ：

- 白/ライトグレーの骨を持つフレンドリーな漫画風スケルトン
- 生き生きとした様子を示すキラキラや点のある大きく表情豊かな眼窩
- 遊び心と社交的な性格を示す広い歯を見せた笑顔
- カジュアルでリラックスしたポーズの関節のある手足
- フレンドリーな手振りやカジュアルなジェスチャーで片手を上げている
- 冗談好きや遊び心のある態度を示す少し傾いた頭
- ライトグレー（#E8E8E8）を主色としたクールで親しみやすいカラーパレット
- 怖くならないように深みを与える柔らかい影とハイライト
- 滑らかで丸みを帯びた骨の形を持つかわいい/カワイイアートスタイル
- 個性を加える小さな蝶ネクタイや帽子などのオプションアクセサリー
- 透明な背景（SVG形式）
- 200x200pxでのWeb表示に適したシンプルでクリーンなデザイン

キャラクターはフレンドリーでユーモラスに見えるべき、ダジャレやジョークを言うのが好きな人のように。全体的な雰囲気は「カジュアルで社交的」- 親しみやすく楽しいことが好きで、決して怖くも威圧的でもない。
```

## ビジュアル要素の詳細

### 色彩
- **メインカラー**: #E8E8E8（ライトグレー/白っぽい骨色）
- **サブカラー**: 白、薄いグレーのグラデーション
- **アクセント**: 黒（眼窩の影）、青や赤（アクセサリー）

### 表情
- 大きな眼窩に小さな光る点（目の代わり）
- 広い笑顔（歯がたくさん見える）
- 片方の眉骨が上がっている（冗談を言っているような）
- リラックスした顎の角度

### ポーズ案
1. 片手を上げて手を振っているポーズ
2. 腕を組んで自信満々に立っているポーズ
3. 指を立てて何かを説明しているポーズ（ダジャレを言う直前のような）

### スタイルガイド
- 骨は丸みを帯びていて柔らかい印象
- 関節部分は明確だが怖くない
- 全体的に明るく清潔感のある色調
- 漫画的でコミカルな雰囲気
- 影は最小限で柔らかく
